# Dauntless

The E-Lab Think Tank Journey Portfolio for **Dauntless** — built for Challenge 06 (Digital Print & Web Design). Documents our team, our E-Lab journey across all six challenges, and our project, **BODA-PAY**: a decentralized cross-border payment platform for the Kigali–Port Sudan trade corridor.

## Stack

Vue 3 · TypeScript · Vite · Tailwind CSS · Vue Router

## Getting started

```bash
npm install
npm run dev
```

```bash
npm run build    # type-check + production build
npm run preview  # preview the production build locally
```

## Project structure

- `src/pages` — the six routes (Home, Team, Culture, Community, Project, Journey)
- `src/components` — shared UI components
- `src/data` — team members and E-Lab challenge content
- `public/team` — team member photos
